# Markdown 記法の紹介

Markdownは軽量マークアップ言語で、簡単に書式付きテキストを作成できます。以下に基本的な記法を紹介します。

## 見出し

```
# 見出し1
## 見出し2
### 見出し3
```

## リスト

- 箇条書き1
- 箇条書き2
  - ネストされた箇条書き

1. 番号付きリスト1
2. 番号付きリスト2

## 強調

```
*イタリック* または _イタリック_
**ボールド** または __ボールド__
```

## リンク

```
[リンクテキスト](https://example.com)
```

## 画像

```
![代替テキスト](画像URL)
```

## コードブロック

```
```
コード例
```
```

## 引用

```
> これは引用です。
```

## テーブル

```
| ヘッダー1 | ヘッダー2 |
| --------- | --------- |
| 内容1     | 内容2     |
```

Markdownを使えば、簡単に整った文書を作成できます！
